# dev-boxes

Vagrant boxes, provisioned via Ansible, for easy setup of local development environments.

Box setups for:

- [Ansible](ansible/Vagrantfile)
- [JHipster](jhipster/Vagrantfile)
- [Ubuntu-Desktop](ubuntu-desktop/Vagrantfile)

## Requirements

- Vagrant 2.0.1+
- VirtualBox 5.1.30+
